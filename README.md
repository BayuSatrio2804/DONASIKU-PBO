##  Deskripsi Proyek  
Repository ini berisi hasil **Tugas Besar Mata Kuliah Pemrograman Berorientasi Objek (PBO)** yang dikembangkan menggunakan **bahasa pemrograman Java**.  
Proyek ini menerapkan konsep-konsep utama **Object-Oriented Programming (OOP)** seperti *encapsulation*, *inheritance*, *polymorphism*, dan *abstraction* dalam pengembangan aplikasi berbasis objek yang modular, efisien, dan mudah dikembangkan.  

---

##  Tujuan  
Tujuan dari proyek ini adalah untuk:
- Mengimplementasikan prinsip dan konsep dasar OOP dalam sistem nyata.  
- Membangun aplikasi dengan struktur kelas yang efisien dan reusable.  
- Melatih kemampuan kolaboratif dalam pengembangan perangkat lunak berbasis tim.

---


##  Konsep OOP yang Diterapkan  
- **Encapsulation:** Melindungi data dengan access modifier dan getter-setter.  
- **Inheritance:** Pewarisan atribut dan method dari superclass ke subclass.  
- **Polymorphism:** Menggunakan *method overriding* dan *method overloading*.  
- **Abstraction:** Menggunakan *abstract class* dan *interface* untuk modularitas desain.
